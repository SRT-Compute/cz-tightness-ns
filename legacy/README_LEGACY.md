# Superseded ingestion
Scripts predating the orientation-verification gate read JHTDB cutouts (delivered with
axis order z, y, x) in the (x, y, z) convention, exchanging derivative directions. All
isotropic quantities produced by those versions are superseded by the code and data in
this deposit. When updating the repository, move any pre-existing analysis scripts into
this folder rather than deleting them: the correction history is part of the record.
